# Resource Limpet

A mod for Oxygen Not Included that attaches to storage buildings and outputs ribbon cable signals for low/high resource levels.

## Features

- Attaches to storage prefabs (gas, liquid, or solid)
- 1x1 size building
- Ribbon cable connector output (uses two channels)
- Sends signal on one channel when storage hits low level
- Sends signal on another channel when storage hits high level
- New building based on Automated Notifier with ribbon input support
- Different notifications depending on channel activation
