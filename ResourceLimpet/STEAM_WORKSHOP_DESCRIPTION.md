[h1]Resource Limpet[/h1]

Attaches to storage buildings and outputs ribbon cable signals for low/high resource levels.

[h2]Features[/h2]
[list]
[*][b]Storage Attachment[/b] - Attaches to gas, liquid, or solid storage buildings
[*][b]Ribbon Cable Output[/b] - Uses two channels for low/high signals
[*][b]Low Level Signal[/b] - Sends signal on one channel when storage hits low level
[*][b]High Level Signal[/b] - Sends signal on another channel when storage hits high level
[*][b]Compact Design[/b] - 1x1 size building
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Build the Resource Limpet[/b] - Place next to a storage building
[*][b]Connect Ribbon Cable[/b] - Connect to your automation network
[*][b]Configure Thresholds[/b] - Set low and high level thresholds
[*][b]Receive Signals[/b] - Automation signals trigger at configured levels
[/olist]

[h3]Tips[/h3]
[list]
[*]Based on the Automated Notifier with ribbon cable support
[*]Different notifications depending on which channel is activated
[*]Great for managing storage levels with automation
[*]Works with all storage building types
[/list]

[h2]Compatibility[/h2]
[list]
[*][b]Oxygen Not Included[/b] - Build 700386 or later
[*][b]Mod API[/b] - Version 2
[*][b]DLC Support[/b] - Works with base game and all DLC
[*][b]Other Mods[/b] - Compatible with most mods
[/list]

[h2]Performance[/h2]
[b]Minimal Performance Impact[/b]
[list]
[*][b]Event-Driven[/b] - Only updates when storage levels change
[*][b]Efficient Checks[/b] - Lightweight threshold comparisons
[*][b]No Polling[/b] - Uses game's built-in storage monitoring
[/list]
