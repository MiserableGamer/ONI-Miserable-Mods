[h1]Controlled Filtering[/h1]

Control which item categories appear as Standard vs Non-Standard in storage filter lists.

[h2]Features[/h2]
[list]
[*][b]Category Control[/b] - Choose which categories are Standard vs Non-Standard
[*][b]Clothing Toggle[/b] - Include/exclude Atmo suits, snazzy suits, etc.
[*][b]Critter Eggs Toggle[/b] - Include/exclude all critter eggs
[*][b]Sublimators Toggle[/b] - Include/exclude Bleach Stone, Oxylite, etc.
[*][b]Bulk Selection[/b] - Make "Select All Standard" work the way you want
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Open Mod Options[/b] - Main Menu ‚Üí Mods ‚Üí Controlled Filtering ‚Üí Options (gear icon)
[*][b]Configure Categories[/b] - Toggle which categories should be Standard vs Non-Standard
[*][b]Restart Game[/b] - Changes require a game restart to take effect
[*][b]Use Storage Filters[/b] - "Select All Standard" will now include your configured categories
[/olist]

[h3]Tips[/h3]
[list]
[*]By default, Clothing, Critter Eggs, and Sublimating items are Non-Standard
[*]Uncheck an option to move that category to the Standard section
[*]This affects all storage buildings (bins, refrigerators, etc.)
[*]Great for quickly selecting eggs or suits with bulk storage operations
[/list]

[h2]Compatibility[/h2]
[list]
[*][b]Oxygen Not Included[/b] - Build 700386 or later
[*][b]Mod API[/b] - Version 2
[*][b]DLC Support[/b] - Works with base game and all DLC
[*][b]Other Mods[/b] - Compatible with mods that don't modify STORAGEFILTERS.SPECIAL_STORAGE
[/list]

[h2]Performance[/h2]
[b]Zero Performance Impact[/b]
[list]
[*][b]Configuration Only[/b] - Changes are applied once at game start
[*][b]No Runtime Overhead[/b] - No continuous processing or monitoring
[*][b]Static Modification[/b] - Simply modifies a static list at startup
[/list]
