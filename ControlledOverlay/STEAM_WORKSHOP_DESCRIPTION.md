[h1]Controlled Overlay[/h1]

Adds a text input field to the Temperature Overlay's relative temperature slider, allowing you to type exact temperature values instead of struggling with the slider.

[h2]Features[/h2]
[list]
[*][b]Exact Temperature Input[/b] - Type the exact temperature you want to focus on
[*][b]Unit Aware[/b] - Respects your temperature unit preference (¬∞C, ¬∞F, K)
[*][b]Replaces Slider Display[/b] - The center temperature display becomes an editable input field
[*][b]Instant Feedback[/b] - Slider snaps to your entered value immediately
[*][b]No Delete Required[/b] - Just start typing to replace the value
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Open Temperature Overlay[/b] - Press F3 or click the temperature button
[*][b]Select Relative Mode[/b] - Choose "Relative Temperature" mode
[*][b]Click Input Field[/b] - Click on the temperature display (center of slider)
[*][b]Type Value[/b] - Just start typing your target temperature
[*][b]Press Enter[/b] - The slider will snap to that temperature
[/olist]

[h3]Tips[/h3]
[list]
[*]No need to delete the existing value - just start typing
[*]Works with any temperature unit you have configured
[*]Great for monitoring specific temperature ranges for crops, critters, or equipment
[*]The overlay will show temperatures relative to your entered value
[/list]

[h2]Compatibility[/h2]
[list]
[*][b]Oxygen Not Included[/b] - Build 700386 or later
[*][b]Mod API[/b] - Version 2
[*][b]DLC Support[/b] - Works with base game and all DLC
[*][b]Other Mods[/b] - Does not conflict with other overlay mods or CapacityControl
[/list]

[h2]Performance[/h2]
[b]Minimal Performance Impact[/b]
[list]
[*][b]Idle[/b] - No performance impact when not using the temperature overlay
[*][b]Active Use[/b] - Lightweight UI updates only when typing
[*][b]Event-Driven[/b] - Only processes input when the overlay is active
[/list]
