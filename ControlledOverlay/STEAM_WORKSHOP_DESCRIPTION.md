[h1]Controlled Overlay[/h1]
Adds a text input field to the Temperature Overlay's relative temperature slider, allowing you to type exact temperature values instead of struggling with the slider.

[h2]Features[/h2]
[list]
[*][b]Exact Temperature Input[/b]: Type the exact temperature you want
[*][b]Unit Aware[/b]: Respects your temperature preference (°C, °F, K)
[*][b]Instant Feedback[/b]: Slider snaps to your entered value immediately
[/list]

[h2]Usage[/h2]
[list]
[*]Open the Temperature Overlay (F3)
[*]Select "Relative Temperature" mode
[*]Click on the temperature input field
[*]Just start typing your target temperature - no need to delete first
[*]Press Enter to snap the slider to your value
[/list]

[h2]Why This Mod?[/h2]
The vanilla temperature slider is hard to fine-tune. This mod lets you type exact values for:
[list]
[*]Monitoring crop temperature requirements
[*]Checking for overheating equipment
[*]Managing critter habitats
[*]Debugging heat transfer issues
[/list]

[h2]Compatibility[/h2]
[list]
[*]Works with base game and all DLC
[*]Does not conflict with other overlay mods
[*]Does not conflict with CapacityControl or other UI mods
[/list]
