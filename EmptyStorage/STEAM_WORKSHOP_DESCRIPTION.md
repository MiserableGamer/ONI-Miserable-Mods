[h1]Empty Storage[/h1]

Adds an "Empty Storage" button to storage buildings, allowing you to quickly drop all contents either immediately or via duplicant labor.

[h2]Features[/h2]
[list]
[*][b]Empty Storage Button[/b] - Adds a convenient button to all storage buildings (gas, liquid, and solid)
[*][b]Immediate Emptying[/b] - Drop all contents instantly with a single click
[*][b]Duplicant Labor Option[/b] - Create a task for duplicants to empty the storage over time
[*][b]Skill Requirements[/b] - Optionally require duplicants to have appropriate skills
[*][b]Configurable Work Time[/b] - Adjustable work time per 100kg (0.1 to 10 seconds)
[*][b]Bionic Dupe Support[/b] - Bionic dupes with Tidying Booster can empty any storage type
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Select Storage Building[/b] - Click on any gas, liquid, or solid storage building
[*][b]Click Empty Storage[/b] - Find the "Empty Storage" button in the building's info panel
[*][b]Contents Drop[/b] - If immediate mode is enabled, contents drop instantly
[*][b]Cancel if Needed[/b] - Click the button again to cancel an in-progress emptying task
[/olist]

[h3]Tips[/h3]
[list]
[*]Immediate emptying is fastest but doesn't require duplicant labor
[*]Skill requirements can be disabled in options if you want any dupe to empty storage
[*]Bionic dupes with Tidying Booster bypass skill requirements
[*]Work time scales with the mass stored - larger containers take longer
[/list]

[h2]Compatibility[/h2]

[*][b]Mod API[/b] - Version 2

[h2]Performance[/h2]
[b]Minimal Performance Impact[/b]
[list]

[*][b]No Background Processing[/b] - No continuous monitoring or polling
[/list]
