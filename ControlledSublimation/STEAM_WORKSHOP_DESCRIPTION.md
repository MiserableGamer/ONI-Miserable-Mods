[h1]Controlled Sublimation[/h1]
Complete control over element sublimation and off-gassing, with separate settings for debris and storage.

[h2]Supported Elements[/h2]
[list]
[*]Bleach Stone → Chlorine
[*]Oxylite → Oxygen
[*]Polluted Dirt → Polluted Oxygen
[*]Slime → Polluted Oxygen
[*]Polluted Water → Polluted Oxygen
[*]Polluted Mud → Polluted Oxygen (DLC)
[/list]

[h2]Features[/h2]
[list]
[*][b]Per-Element Control[/b]: Configure each element independently
[*][b]Debris vs Storage[/b]: Separate toggles for each context
[*][b]Rate Multiplier[/b]: Adjust speed from 0% to 1000%
[*][b]Oxysconce Compatible[/b]: Bleach Stone Sconce always works
[*][b]DLC Safe[/b]: Works with or without DLC installed
[/list]

[h2]Configuration[/h2]
Access mod options through the game's Mod Options menu.

Each element has three settings:
[list]
[*][b]As Debris[/b]: Sublimate when lying on ground
[*][b]In Storage[/b]: Sublimate when in containers
[*][b]Rate Multiplier[/b]: Speed adjustment (0.0 - 10.0)
[/list]

[b]Note:[/b] Changes require a game restart.

[h2]Compatibility[/h2]
[list]
[*]Works with all DLC
[*]DLC elements safely ignored if DLC not installed
[*]Compatible with most mods
[*]No known conflicts
[/list]
