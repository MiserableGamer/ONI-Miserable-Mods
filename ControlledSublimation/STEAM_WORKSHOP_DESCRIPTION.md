[h1]Controlled Sublimation[/h1]

Complete control over element sublimation and off-gassing, with separate settings for debris and storage.

[h2]Features[/h2]
[list]
[*][b]Per-Element Control[/b] - Configure each element independently
[*][b]Debris vs Storage[/b] - Separate toggles for debris on ground vs items in containers
[*][b]Rate Multiplier[/b] - Adjust sublimation speed from 0% to 1000%
[*][b]Oxysconce Compatible[/b] - Bleach Stone Sconce always works regardless of settings
[*][b]DLC Safe[/b] - Works with or without DLC installed
[/list]

[h2]Supported Elements[/h2]
[list]
[*]Bleach Stone ‚Üí Chlorine
[*]Oxylite ‚Üí Oxygen
[*]Polluted Dirt ‚Üí Polluted Oxygen
[*]Slime ‚Üí Polluted Oxygen
[*]Polluted Water ‚Üí Polluted Oxygen (bottles)
[*]Polluted Mud ‚Üí Polluted Oxygen (DLC)
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Open Mod Options[/b] - Main Menu ‚Üí Mods ‚Üí Controlled Sublimation ‚Üí Options (gear icon)
[*][b]Configure Elements[/b] - For each element, set:
[list]
[*]As Debris - Sublimate when lying on ground
[*]In Storage - Sublimate when in containers
[*]Rate Multiplier - Speed adjustment (0.0 - 10.0)
[/list]
[*][b]Restart Game[/b] - Changes require a game restart to take effect
[/olist]

[h3]Tips[/h3]
[list]
[*]Set rate to 0.0 to completely disable sublimation for an element
[*]Oxysconce (Bleach Stone Sconce) always emits chlorine regardless of settings
[*]DLC elements are safely ignored if the DLC is not installed
[*]Use storage settings to prevent off-gassing in bins while allowing debris to emit
[/list]

[h2]Compatibility[/h2]
[list]
[*][b]Oxygen Not Included[/b] - Build 700386 or later
[*][b]Mod API[/b] - Version 2
[*][b]DLC Support[/b] - Works with base game and all DLC
[*][b]Other Mods[/b] - Compatible with most mods, no known conflicts
[/list]

[h2]Performance[/h2]
[b]Minimal Performance Impact[/b]
[list]
[*][b]Configuration Only[/b] - Settings applied at game start
[*][b]Lightweight Patches[/b] - Only modifies sublimation rate calculations
[*][b]No Polling[/b] - Event-driven, not continuous monitoring
[/list]
