[h1]Copy Tile Tool[/h1]

Replace tiles in bulk by selecting a destination type, then a source type to replace, then drag-selecting an area.

[h2]Features[/h2]
[list]
[*][b]Tile Type Replacement[/b] - Change tiles from one type to another (e.g., Standard Tile ‚Üí Mesh Tile)
[*][b]Exact Matching[/b] - Only replaces tiles with the same type AND material as your source
[*][b]Bulk Replacement[/b] - Drag to select multiple tiles at once
[*][b]Safe Replacement[/b] - Queues deconstruction and reconstruction through normal game systems
[*][b]All Tile Types[/b] - Works with Standard, Mesh, Airflow, Insulated, Metal, Plastic, Bunker, Carpet, Window, and all DLC tiles
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Select Destination Tile[/b] - Click on a tile that represents what you want tiles to become
[*][b]Click "Copy Tile" Button[/b] - In the tile's side screen, click the "Copy Tile" button
[*][b]Select Source Tile[/b] - Click on a tile of a DIFFERENT type that you want to replace
[*][b]Drag to Apply[/b] - Drag over the area to queue replacements for all matching tiles
[/olist]

[h3]Tips[/h3]
[list]
[*]This tool is for changing tile [b]types[/b] (e.g., Standard Tile ‚Üí Mesh Tile)
[*]For changing tile [b]materials[/b] only (e.g., Sandstone Tile ‚Üí Granite Tile), use the [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3626072188]Copy Materials Tool[/url] instead ([url=https://github.com/MiserableGamer/ONI-Miserable-Mods/tree/master/CopyMaterialsTool]GitHub[/url])
[*]Only tiles that exactly match the destination (both type AND material) will be replaced
[*]If the source and destination are the same tile type, the tool will reject the selection
[/list]
