[h1]Bonbon Tree Boost[/h1]
Enables the editing of the Bonbon tree, growth, production, and amount of fertilizer needed for domesticated plants.

[h2]Features[/h2]
[list]
[*][b]Configurable Nectar Production[/b] - Adjust the multiplier for Space Tree sugar water (nectar) production (0.1x to 10x)
[*][b]Growth / Branch Efficiency Control[/b] - Control how quickly Space Trees reach optimal production (0.1x to 5x)
[*][b]Fertilizer Consumption Adjustment[/b] - Modify how much Snow fertilizer the trees consume (0x to 5x)
[*][b]Fully Configurable[/b] - All settings are adjustable through the mod options menu
[*][b]Works with Existing Saves[/b] - Changes apply to existing trees, new trees, and all saves
[/list]

[h2]Configuration Options[/h2]
The mod includes three configurable options:
[olist]
[*][b]Nectar Production Multiplier[/b] (Default: 2.0x)
[list]
[*]Multiplier for Space Tree sugar water production
[*]Range: 0.1 to 10.0
[*]Higher values = more nectar production
[/list]
[*][b]Growth / Branch Efficiency Multiplier[/b] (Default: 1.0x)
[list]
[*]Affects how quickly Space Trees reach optimal production
[*]Range: 0.1 to 5.0
[*]Higher values = trees reach optimal production faster
[/list]
[*][b]Fertilizer Consumption Multiplier[/b] (Default: 1.0x)
[list]
[*]Multiplier for Snow fertilizer consumption
[*]Range: 0.0 to 5.0
[*]Set to 0.0 to disable fertilizer consumption entirely
[/list]
[/olist]

[h2]How It Works[/h2]
The mod modifies the Space Tree's production mechanics by adjusting:
[list]
[*][b]Production Duration[/b]: Lower duration means more frequent production cycles
[*][b]Branch Requirements[/b]: Adjusts how many branches are needed for optimal production
[*][b]Fertilizer Consumption Rate[/b]: Directly modifies the mass consumption rate of Snow fertilizer
[/list]

Changes are applied when trees are created, affecting both existing and new trees in your save.

[h2]Compatibility[/h2]
[list]
