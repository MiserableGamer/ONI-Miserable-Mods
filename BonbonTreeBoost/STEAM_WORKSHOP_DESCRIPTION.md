[h1]Bonbon Tree Boost[/h1]

Configure the Bonbon Tree (Space Tree) nectar production, growth speed, and fertilizer consumption to suit your gameplay needs.

[h2]Features[/h2]
[list]
[*][b]Separate Wild and Domestic Settings[/b] - Configure wild and domesticated trees independently
[*][b]Configurable Nectar Production[/b] - Adjust multipliers for Space Tree sugar water production (1.0x to 10.0x)
[*][b]Independent Growth Rate Control[/b] - Control trunk and branch growth rates separately (0.1x to 10.0x)
[*][b]Fertilizer Consumption Adjustment[/b] - Modify Snow fertilizer consumption for domesticated trees (0.0x to 10.0x)
[*][b]Production Balance Control[/b] - Choose whether wild or domestic trees produce more
[*][b]Branch Harvesting Control[/b] - Enable or disable wood harvesting from branches
[*][b]Works with Existing Saves[/b] - Changes apply to existing trees, new trees, and all saves
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Open Mod Options[/b] - Main Menu ‚Üí Mods ‚Üí Bonbon Tree Boost ‚Üí Options (gear icon)
[*][b]Configure Wild Trees[/b] - Set nectar production, trunk growth, and branch growth multipliers
[*][b]Configure Domesticated Trees[/b] - Set nectar production, growth rates, and fertilizer consumption
[*][b]Set Production Balance[/b] - Choose Wild Advantage, Equal, or Domestic Advantage
[*][b]Restart Game[/b] - Changes require a game restart to take effect
[/olist]

[h3]Tips[/h3]
[list]
[*]Set fertilizer consumption to 0.0 to disable fertilizer requirements for domesticated trees
[*]Use the Production Balance setting to control which tree type produces more
[*]Branch harvesting can be disabled if you only want nectar production
[*]Growth rate changes affect how quickly trees mature
[/list]
