[h1]No Disease[/h1]
Completely disables the disease system in Oxygen Not Included - not just immunity, but removes disease mechanics entirely from the game.

[b][i]Note: This mod is intended as an interim solution until [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1967921388]FastTrack[/url] by Peter Han is updated for the current game version. If you only want the "No Disease" functionality without FastTrack's other features, this standalone mod is for you. This mod will be removed from the workshop if requested by other modders.[/i][/b]

[h2]Features[/h2]
This mod comprehensively disables all disease-related systems:
[list]
[*][b]No Germs[/b] - All existing germs are wiped from the world on load
[*][b]No Disease Spread[/b] - Germs cannot spread through air, water, or pipes
[*][b]No Infections[/b] - Duplicants cannot get sick from germs
[*][b]No Disease UI[/b] - Disease overlay and panels are hidden
[*][b]No Disease Tools[/b] - Disinfect tool is removed
[*][b]Hidden Buildings[/b] - Disease-related buildings are deprecated:
[list]
[*]Doctor Stations (basic and advanced)
[*]Disease Sensors (gas, liquid, solid conduits)
[*]Apothecary buildings (except when Radiation DLC is enabled - kept for rad pills)
[/list]
[/list]

[h2]Why Use This Mod?[/h2]
[list]
[*][b]Performance[/b] - Disease simulation can impact game performance on large colonies
[*][b]Gameplay Preference[/b] - Some players find the disease system tedious rather than fun
[*][b]Simplification[/b] - Focus on other aspects of colony management
[/list]
