[h1]Capacity Control[/h1]

Increases the storage capacity text input character limit beyond the vanilla 6 characters, allowing you to type larger storage values.

[h2]Features[/h2]
[list]
[*][b]Configurable Character Limit[/b] - Choose how many extra characters to add (1-10) via mod options
[*][b]Works with All Storage[/b] - Applies to any building that uses the capacity control side screen
[*][b]Compatible with Storage Mods[/b] - Designed to work alongside mods that increase storage capacity
[/list]

[h2]How to Use[/h2]
[olist]
[*][b]Open Mod Options[/b] - Main Menu ‚Üí Mods ‚Üí Capacity Control ‚Üí Options (gear icon)
[*][b]Set Additional Characters[/b] - Choose how many extra characters beyond the vanilla 6 limit
[*][b]Restart Game[/b] - Changes require a game restart to take effect
[*][b]Type Larger Values[/b] - You can now type larger storage values in any capacity control
[/olist]

[h3]Tips[/h3]
[list]
[*]Default adds 2 extra characters (8 total), allowing values up to 9,999,999 kg
[*]Useful when using mods that increase storage capacity beyond 999,999 kg
[*]The slider can set values beyond the text limit, but typing requires this mod
[/list]

[h2]Compatibility[/h2]
[list]
[*][b]Oxygen Not Included[/b] - Build 700386 or later
[*][b]Mod API[/b] - Version 2
[*][b]DLC Support[/b] - Works with base game and all DLC
[*][b]Other Mods[/b] - Compatible with mods that increase storage capacity
[/list]

[h2]Performance[/h2]
[b]Zero Performance Impact[/b]
[list]
[*][b]One-Time Modification[/b] - Only changes the input field limit once at screen initialization
[*][b]No Runtime Overhead[/b] - No continuous processing or monitoring
[*][b]Lightweight Patch[/b] - Single Harmony patch on UI initialization
[/list]

[h2]Future Updates[/h2]
